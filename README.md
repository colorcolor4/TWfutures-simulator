## 怎麼用

頁面有兩個分頁：**回測**（設策略、跑、看結果）、**比較**（多策略橫向比對）。

### 一、快速開始
按頁面上方任一**預設**鈕帶入一組範例參數 → 填**本金** → 按 **回測** → 看結果。想自訂就照下面改參數。

### 二、設定策略（回測分頁）

**① 本金與上限**
- **本金**：起始資金（元）。
- **cap（槓桿上限）**：總曝險天花板，留空＝取曲線最大值。安全閥，防加碼加過頭。

**② 買進策略 ＝ 目標曝險曲線**（本工具核心）
一張表，每列＝「**大盤自高點跌 X% → 目標槓桿 Y 倍**」。程式每天看大盤位置落在哪一格，就把部位調到對應倍數——跌越深、槓桿越高（低接），回到高點自動降回基礎倍數。
- 預設第一列 `跌 0% → 1 倍`＝高點時的基礎曝險。
- 按「＋加一格」往下加，例：`0→1｜6→1.25｜10→1.5｜20→2`（大盤跌 20% 時上到 2 倍）。
- 「－移除」刪該格。

**③ 賣出策略**（勾選＝啟用，可複選）
- **權益停損**：權益從高點回撤達 X% → 全平。
- **創新高後跌破 N 日線出場**：指數創新高後改抱到跌破均線才獲利了結。
- **快慢線死叉出場**：快線下穿慢線出場（例 快 5 / 慢 10）。
- **單日跌幅 ≥ X% 出場**：單日大跌保護。
- **盤整 N 日出場**：均線走平、久盤無進展就出。

**④ 進階（power user，可摺疊）**
- **探底 probe / hybrid**：深水區改用「破前低停損、彈回再試」的探底式低接。
- **濾網**：bear_filter（空頭段加碼減半）、requick、confirm_ma（右側確認才成交）、vol_gate（波動不夠不加碼）。
- **馬丁格爾 tranches_pts**：每跌 N 點加碼 Y 倍（與曲線二選一，勾了就用點數批次）。

### 三、看結果（回測分頁）

- **KPI 卡**：終值、總報酬、最大回撤、強平次數、追繳警告、**會不會爆倉**（被強平＝💀曾強平N次／權益歸零＝💀帳戶歸零／都沒有＝✅未爆倉）、**最佳年**、**最差年**。
- **倍數 vs 口數對照**：理論倍數模型 vs 含成本的口數真實績效（口數必然略低，這才是實盤）。
- **權益曲線**、**年度報酬表**、**追繳／強平事件清單**。

### 四、比較多個策略（比較分頁）

回測跑完按 **存進比較** → 切到**比較**分頁，每個策略一列橫向並排（總報酬／回撤／強平／最佳最差年／爆倉）。
- 結果存在瀏覽器本機（localStorage），關掉頁面再開仍在。
- 勾選要刪的列 → **刪除勾選**。
- 點策略名 → 載回回測分頁的表單，可微調重跑。
